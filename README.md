# stats507-coursework

This repository contains coursework for **STATS 507**, a course at the University of Michigan focusing on Python programming and data analysis techniques. The course covers topics such as:

- Python fundamentals (variables, control flow, functions, recursion)
- Data manipulation and exploration
- Numerical computation and approximations
- Writing reusable and well-documented Python code

The purpose of this repository is to organize and showcase assignments, exercises, and projects completed during the course, demonstrating both programming proficiency and applied data analysis skills.
